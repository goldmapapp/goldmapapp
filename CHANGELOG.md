# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-04

### Added
- Initial project setup with Flutter 3.38.9 and Dart 3.10.8
- Multi-platform support (Android, iOS, Web, Windows)
- State management with Flutter Riverpod 2.x
- Navigation with GoRouter
- Dependency injection with GetIt + Injectable
- HTTP client with Dio
- Local storage with Hive and SharedPreferences
- Code generation setup (Freezed, JSON Serializable)
- Comprehensive linting with Very Good Analysis
- Clean architecture folder structure
- Core utilities and error handling
- Build scripts and Makefile
- VS Code configuration with recommended extensions
- MCP servers configuration for enhanced development
- Testing setup with Mockito
- UI components (SVG support, cached images, shimmer effects)
- Environment-based configuration (dev, staging, production)
- Logger utility for debugging
- Comprehensive README documentation
- Git configuration with proper .gitignore

### Development Tools
- Build runner for code generation
- Very Good Analysis for code quality
- Formatted with 120 character line length
- Automatic import organization
- Multiple launch configurations for different environments
